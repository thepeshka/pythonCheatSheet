# Оператор ветвления `if`

# Синтаксис:
```python
if <логическое выражение>:
    <блок кода, который выполняется, если выражение истинно>
else:
    <блок кода, который выполняется, если выражение ложно>
```
```python
if <логическое выражение>:
    <блок кода, который выполняется, если выражение истинно>
```
# Примеры:
```python
a = 5
b = 6
if a > b:
    print("a больше, чем b")
else:
    print("a меньше или равно b")
```

a = 5
b = 6
